# CIS*2750 Assignment 3 - Luca Bozzetto



## Installation and Running

### 1. Compile Library

```Bash
# From /parser directory
make
```

### 2. Install

```Bash
# From the root of the directory
npm install
```

### 3. Running Server

```Bash
npm run dev PORT
# Server will be accessible at http://localhost:PORT
```
