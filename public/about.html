VCard Viewer

A WebApp to upload, parse and inspect vCards!

This website was developed as part of the CIS*2750 course @ University of Guelph. The entire source code will be available once the course is completed on my Github page.

The files parser is written in C and is located in the 'parser' directory. Please mind that the course assignments simplified the parsing so at the moment not all valid vCards files will be successfully processed. In the future I might add functionalities and fully implement a parser.
